🎌 Kanji Helper Extension Test Page

How to test the Kanji Helper extension:

  1. Install the extension in Chrome
  2. Select any Japanese text below
  3. Right-click and choose "Translate with Kanji Helper"
  4. Or use the extension popup to translate text

Sample Japanese Text to Translate:

Basic Kanji:
日本人は漢字を学校で学びます。
Simple Words:
大きい山と小さい川があります。
Time and Date:
今日は月曜日です。時間は午前十時です。
Individual Characters:
水 火 土 木 金 学 生 先 年

Instructions:

Try selecting different parts of the Japanese text above:

  • Select individual kanji characters like: 日, 本, 人, 漢, 字
  • Select words like: 日本, 学校, 時間
  • Select phrases like: 日本人, 漢字を学ぶ

The extension will show translations for each character with readings and meanings!
