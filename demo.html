🚀 How to Use:

  1. Install the extension in Chrome (chrome://extensions/ → Load unpacked)
  2. Select Japanese text on this page or any webpage
  3. Right-click and choose "Translate with Kanji Helper"
  4. View translations in the beautiful popup that appears
  5. Use the extension popup to translate your own text
